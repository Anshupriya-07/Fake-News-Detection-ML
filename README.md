# 📰 Fake News Detection using Machine Learning

This project detects whether a news article is **Fake or Real** using Natural Language Processing and Machine Learning techniques.

---

## 🚀 Features

• Text preprocessing using NLP  
• TF-IDF feature extraction  
• Multiple ML models  
• Model comparison  
• Cross-validation  
• Interactive web interface using Streamlit  

---

## 🤖 Models Used

| Model | Accuracy |
|------|------|
| Naive Bayes | 93.4% |
| Decision Tree | 99.7% |
| Logistic Regression | 98.8% |

Decision Tree achieved the best performance.
